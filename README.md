## What is Amigacoin?
Amigacoin is another open source peer-to-peer Litecoin-derived cryptocurrency.

 - 2 minute block targets
 - 2 billion total coins
 - 100 to 1000 random coins per block
 - 1440 blocks to retarget difficulty
